# MAX Channel -> Comments Bot

Бот для MAX, который автоматически превращает посты канала в обсуждение в чате комментариев.

## Что делает бот

- Слушает события через webhook (`/webhook`).
- Для новых постов в подключенных каналах:
  - пересылает контент в чат комментариев;
  - добавляет под постом кнопки (в чат и к сообщению).
- Даёт админ-панель через `/admin`:
  - привязка `канал -> чат комментариев`;
  - настройка текстов кнопок и рекламной ссылки;
  - mute по времени;
  - просмотр и редактирование текста/картинки постов.

## Коротко в двух словах

Это **бот-ретранслятор комментариев**: берёт посты из канала, отправляет их в чат обсуждения и помогает админам управлять этим процессом.

## Быстрый запуск

1. Скопируйте `.env.example` в `.env` и заполните:
   - `MAX_BOT_TOKEN`
   - `ADMIN_USER_IDS`
   - `WEBHOOK_URL`
2. Поднимите контейнер:
   ```bash
   docker compose up -d --build
   ```
3. Проверьте:
   - `GET /health` через ваш reverse proxy.

## Основные переменные

- `MAX_BOT_TOKEN` — токен бота MAX.
- `WEBHOOK_URL` — публичный HTTPS URL вебхука.
- `WEBHOOK_LISTEN` — адрес/порт приложения внутри контейнера (по умолчанию `0.0.0.0:8000`).
- `WEBHOOK_SECRET` — опциональный секрет для заголовка `X-Max-Bot-Api-Secret`.
- `ADMIN_USER_IDS` — user_id админов через запятую.
